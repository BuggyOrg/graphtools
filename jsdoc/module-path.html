(static) equal(path1, path2) → {boolean}

Tests if the two pathes are equal (start and end in the same nodes and go through the exact same nodes on their way). It does not check if the taken ports are always the same.

Parameters:
Name Type Description
path1 Path

A path as an array of strings or an array of {node: name, ...}.
path2 Path

The path to compare path1 to. Must have the same format as path1.
