(static) walk(graph, node, path, optionsopt) → {Array.<string>|Array.<object>}

Tries to follows a given path through the graph starting at node. It uses the successor function to track the neighbors.

Parameters:
Name Type Attributes Default Description
graph Graphlib

The graph
node string | Object

The node can either be

  • the name of the starting node
  • an object {node: 'START', port: 'USE_PORT'} that forces the walk to use USE_PORT for the first node.
path Array.<string> | function

The path itself can be

  • an array of ports that should be followed (even if the USE_PORT field is set, you must start with it as the first port). If it is not possible to follow the port a empty list of paths is returned.

  • a function that is called for every node on the path and that returns

    • a list of ports to continue with
    • an empty list to stop the walk
    • null to discard the branch.

    The function takes three arguments: graph, node, port, where the node is the name of the current node on the path. The port is the port it used to get to node.
options Object <optional>
 {keepPorts: false}

An optional object that can have the following properties

  • keepPorts: If this field is true, walk will return a list of objects that each have the format: {node: 'NODE_ID', port: 'INPUT_PORT', edge: EDGE}. It will not have an edge object for the first node on the path.
Source:

Returns:

It returns the list of nodes on the path.
